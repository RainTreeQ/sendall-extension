## 📤 Chrome Web Store 提交流程

### 步骤 1: 登录开发者控制台
访问: https://chrome.google.com/webstore/devconsole

### 步骤 2: 创建新项目
1. 点击"新建项目"
2. 选择"上传 ZIP 文件"
3. 上传 `release/sendol-extension-v0.1.0.zip`

### 步骤 3: 填写商店信息
```
名称: AI Broadcast - 多屏同步发送
版本: 0.1.0
类别: 生产力工具
语言: 支持多语言（en, zh_CN）
```

### 步骤 4: 上传素材
- [ ] 图标 (128x128)
- [ ] 截图 (至少 1 张，推荐 3-5 张)
- [ ] 宣传图片 (可选，440x280)

### 步骤 5: 隐私与合规
- [ ] 隐私政策 URL: （你的网站/privacy）
- [ ] 权限说明已填写
- [ ] 勾选"我已阅读并同意开发者协议"

### 步骤 6: 提交审核
- [ ] 点击"提交审核"
- [ ] 选择"公开"或"不公开"
- [ ] 确认提交
